Automatically eliminate attached media from posts and featured images uploaded via Media button.

== Description ==

- Are you worrying about the issue of growing database or memory that is caused by the existence of unnecessary images or metabox of removed posts?

- Now Sky Remove Attached Files And Featured Images Automatically plugin helps you solve this problem. Moreover, this tool is also used for custom post type. Before you remove posts, images uploaded via "Add Media" will automatically be eliminated. Besides, all metaboxes and featured images of those posts  will also be removed quickly

- So Why don`t you try activating the plugin and rate 5  stars if it avails.

Forum: http://forum.skygame.mobi

== Installation ==

1. Upload `sky-auto-remove-attached-media.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. That's all. It would use out of the box, no settings needed. If you want to disable the functionality, disable the plugin from the Plugins section.
